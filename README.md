## Project Overview
Mood & Bites Cafe is a cozy caf&eacute; located at 83 National Road, Barangay Putatan, Muntinlupa City. This website aims to establish an online presence for the cafe, showcase their menu, provide operating hours and location information, and allow customers to make enquiries about catering and events.

The website consists of 5 core pages:
- Homepage (index.html)
- About Us (about.html)
- Services/Menu (services.html)
- Enquiry (enquiry.html)
- Contact (contact.html)

## Website Goals and Objectives

### Goals
1. Establish an online presence for Mood & Bites Cafe
2. Showcase menu items with prices (converted from PHP to ZAR)
3. Provide clear location and operating hour information
4. Generate customer enquiries through an online form

### Key Performance Indicators (KPIs)
1. Number of enquiry form submissions per month
2. Page views per week
3. User engagement time on menu page

## Key Features and Functionality

| Feature | Description |
| :--- | :--- |
| Responsive Navigation Menu | Links to all 5 pages, works across all devices |
| Hero Image Section | Welcoming image from cafe's Facebook page |
| Menu/Products Page | List of food and beverage items with prices |
| Enquiry Form | For catering and event enquiries |
| Contact Form | General message form that compiles into email |
| Google Maps Integration | Showing both Putatan and Alabang locations |
